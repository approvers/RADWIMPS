## ルール
- `then().then().then().世()`のようなメソッドチェーンで "前前前世" と末尾に改行付きで出力すればOKです。
- `Promise`や`Future`等の言語機能があればぜひ活用してください。

 
なお、このレポジトリは、実在するバンド「RADWIMPS」とは何ら関係ありません。
